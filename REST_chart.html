RESTful Routes

A table of all 7 RESTful routes

Name Path HTTP Verb Purpose Mongoose Method
Index /items GET List all items Item.find()
New /items/new GET Show new item form N/A
Create /items POST Create a new item, then redirect somewhere Item.create()
Show /items/:id GET Create a new item, then redirect somewhere Item.findById()
Edit /items/:id/edit GET Create a new item, then redirect somewhere Item.findById()
Update /items/:id PUT Create a new item, then redirect somewhere Item.findByIdAndUpdate()
Destroy /items/:id DELETE Create a new item, then redirect somewhere Item.findByIdAndRemove()
